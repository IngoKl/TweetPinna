TweetPinna - Twitter Status Archiver.
 
TweetPinna streams Twitter statuses into a
MongoDB database based on given search terms.
 
Author: Ingo Kleiber <ingo@kleiber.me> (2017)
License: MIT
Version: 1.0.1
